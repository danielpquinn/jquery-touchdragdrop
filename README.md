jquery-touchdragdrop
====================

A simple touch friendly drag and drop jQuery plugin.

## Markup

```html
<div class="drop">
  <div class="item"><img src="http://placekitten.com/60/60" /></div>
  <div class="item"><img src="http://placekitten.com/60/60" /></div>
  <div class="item"><img src="http://placekitten.com/60/60" /></div>
</div>
<div class="drop">
  <div class="item"><img src="http://placedog.com/60/60" /></div>
  <div class="item"><img src="http://placedog.com/60/60" /></div>
  <div class="item"><img src="http://placedog.com/60/60" /></div>
</div>
```

## Javascript

```javascript
$('.item').touchdragdrop();
```
